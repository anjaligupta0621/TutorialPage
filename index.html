Introduction

Hi All, let's start with what HTML is.

HTML is Hyper Text Markup Language and it provides a skeleton to any website. All the websites that we see in our browsers are written in HTML.

HTML contains tags which specify which line is of which type. These tags are enclosed in '<>'.

Elements that I have learned

How to write Paragraphs.: Paragraphs can be written by using p tag.

How to write headers: There are 6 types of headers that can be written in HTML. They are h1, h2, h3, h4, h5 and h6. The text enclosed in the h1 tag will be the largest and that enclosed in the h6 tag will be the smallest.

How to style fonts: The fonts can be made bold, italic and can be underlined too. The text can be made bold by enclosing it within b tag, italic when enclosed in i tag and underlined when enclosed in u tag.
